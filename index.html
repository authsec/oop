OBEX Object Passing

This site contains information as well as JavaDoc for the OBEX Object Passing library API.

This library is a result of a diploma thesis which is available as (in german) in the following formats

  • PDF (A4)
  • PDF print version (A4)
  • PDF print version (A5 Booklet)
  • HTML
  • HTML (single page)

This library can be used to transfer Java objects over a Bluetooth connection nearly automatically. The library attemps to find it's communication partner on it's own, establishes a connection and transfers the wanted information to the remote bluetooth device.
